# Simple Portfolio

- 軽量、高速なウェブサイト。
- あなたのお好みに簡単カスタマイズ。
- 数クリック、そして無料で作成可能。

[いますぐデプロイ](example.com)

## 仕組み・コンセプト

- モバイルファーストデザイン
- シンプルにあなたを表現する
  - 自己紹介ページで簡潔に伝えよう
  - Works ページであなたが生み出したものを見せつけよう。
  - Blog ページで共感を生み出そう。
  - Contact ページでたくさんの人に見つけてもらおう

## 使用方法
